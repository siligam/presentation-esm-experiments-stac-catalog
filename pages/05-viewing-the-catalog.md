# Viewing the Catalog

The STAC catalog service runs at AWI, behind the VPN

1. **Your experiment runs** on Levante (DKRZ) → `catalog.duckdb` is created automatically
2. **Share** your `catalog.duckdb` with the catalog administrator*
3. The catalog administrator **registers** it with the running catalog server — no restart needed
4. **Browse** at the catalog web interface
