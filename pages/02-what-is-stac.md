# What is STAC?

**SpatioTemporal Asset Catalog** — a standard for describing geospatial datasets

- **Standardized metadata** — common structure for any climate model output
- **Searchable** — filter by variable, time range, model component, namelist parameter
- **Self-contained** — metadata stored in a single DuckDB file per experiment
- **Federated** — multiple experiment databases served as one unified catalog

<div class="mt-8 p-4 bg-blue-50 dark:bg-blue-900 rounded">
💡 Instead of navigating file systems, researchers can <b>query</b> experiment output like a database.
</div>
